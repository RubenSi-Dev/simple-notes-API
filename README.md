# simple-notes-API
## Goal: Learn how to build and expose an HTTP API using Go.

### Features

- GET /notes → returns a list of notes

- POST /notes → create a new note

- DELETE /notes/:id → delete a note

- Store notes in memory (no database yet)

### Skills

- How HTTP routes work

- Parsing JSON requests and returning JSON responses

- How to run and test a Go server

- Basic REST API design

- Error handling

- Using Postman or curl to test endpoints
